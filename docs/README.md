# Dokumentasi Sistem Pengamanan Kesehatan MBG

Folder ini berisi dokumentasi lengkap untuk sistem pengamanan kesehatan kegiatan MBG, termasuk SOP (Standard Operating Procedure) dan formulir standar.

## Daftar Dokumen

### SOP (Standard Operating Procedure)

1. **SOP-Skrining-Kesehatan.md**
   - Prosedur skrining kesehatan pra-kegiatan
   - Kriteria penilaian (Fit/Fit with Notes/Not Fit)
   - Penerbitan Health Clearance Pass

2. **SOP-Entry-Check.md**
   - Prosedur pemeriksaan di pintu masuk/checkpoint
   - Rapid triage (Hijau/Kuning/Merah)
   - Validasi Health Clearance dengan QR Code

3. **SOP-Penanganan-Insiden.md**
   - Klasifikasi insiden (Ringan/Sedang/Berat)
   - Prosedur respons medis
   - Kode darurat (Code Blue, Code Red, dll)
   - Dokumentasi insiden

4. **SOP-Rujukan-RS.md**
   - Kriteria dan prosedur rujukan
   - Daftar rumah sakit rujukan
   - Format surat rujukan
   - Koordinasi dan follow-up

### Formulir

5. **Form-Skrining-Kesehatan.md**
   - Template formulir registrasi dan skrining kesehatan
   - Data personel
   - Riwayat kesehatan
   - Hasil pemeriksaan
   - Pernyataan dan persetujuan

## Cara Penggunaan

### Untuk Petugas Kesehatan
1. Baca dan pahami semua SOP sebelum kegiatan
2. Ikuti prosedur sesuai SOP yang berlaku
3. Gunakan formulir standar untuk dokumentasi
4. Laporkan setiap penyimpangan atau kejadian khusus

### Untuk Personel
1. Baca informasi tentang prosedur skrining kesehatan
2. Siapkan dokumen dan informasi kesehatan yang diperlukan
3. Ikuti instruksi petugas kesehatan
4. Laporkan segera jika ada keluhan kesehatan

### Untuk Pimpinan/Komando
1. Pastikan semua petugas memahami SOP
2. Monitor implementasi prosedur
3. Review laporan berkala
4. Evaluasi dan perbaikan berkelanjutan

## Prinsip Utama

### Keamanan (Safety First)
- Kesehatan dan keselamatan personel adalah prioritas utama
- Tidak ada kompromi dalam standar kesehatan
- Deteksi dini dan pencegahan

### Kecepatan (Rapid Response)
- Respons cepat untuk setiap insiden
- Prosedur yang efisien
- Koordinasi yang baik

### Dokumentasi (Accountability)
- Semua kegiatan terdokumentasi dengan baik
- Traceability untuk evaluasi
- Pertanggungjawaban yang jelas

### Komunikasi (Clear Communication)
- Jalur komunikasi yang jelas
- Informasi akurat dan tepat waktu
- Koordinasi antar tim
