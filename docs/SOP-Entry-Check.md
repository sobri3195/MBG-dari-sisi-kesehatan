# SOP SECURITY CHECK KESEHATAN DI GATE

## 1. Tujuan
Memastikan hanya personel yang sehat yang diizinkan memasuki area kegiatan MBG untuk mencegah penyebaran penyakit dan risiko kesehatan.

## 2. Ruang Lingkup
SOP ini berlaku untuk semua personel yang akan memasuki area kegiatan MBG melalui checkpoint/gate yang telah ditentukan.

## 3. Lokasi Pelaksanaan
- Gate utama kegiatan
- Checkpoint tambahan sesuai kebutuhan
- Pos kesehatan pintu masuk

## 4. Prosedur

### 4.1. Persiapan Pos Kesehatan
1. Setup pos minimal 2 jam sebelum kegiatan
2. Peralatan yang harus tersedia:
   - Thermometer gun (infrared)
   - Tensimeter digital
   - Pulse oximeter
   - Scanner QR Code
   - Tablet/laptop untuk input data
   - Formulir manual (backup)
   - Hand sanitizer dan APD

### 4.2. Validasi Health Clearance
1. Personel menunjukkan Health Clearance Pass (QR Code)
2. Petugas scan QR Code untuk verifikasi:
   - Keaslian clearance
   - Status: Valid/Expired/Revoked
   - Data personel
   - Status kebugaran terakhir

3. Jika tidak ada QR Code:
   - Cari manual di database berdasarkan nama/ID
   - Jika tidak terdaftar: arahkan ke registrasi darurat

### 4.3. Pemeriksaan Cepat (Rapid Check)
1. Pengukuran suhu tubuh (wajib):
   - Normal: <37.5°C
   - Perhatian: 37.5-38°C
   - Demam: >38°C

2. Observasi visual:
   - Batuk berat atau bersin-bersin
   - Sesak napas
   - Tampak lemas/pucat
   - Berkeringat berlebihan

3. Tanya keluhan:
   - "Apakah Anda merasa tidak enak badan?"
   - "Ada keluhan pusing, mual, atau nyeri?"

### 4.4. Kategori Triage

#### HIJAU (Boleh Masuk)
- Suhu normal (<37.5°C)
- Tidak ada gejala
- Clearance valid
- Kondisi umum baik
- **KEPUTUSAN: APPROVED**

#### KUNING (Perlu Perhatian)
- Suhu sedikit tinggi (37.5-38°C)
- Keluhan ringan (lelah, sedikit pusing)
- Perlu observasi lebih lanjut di poskes
- **KEPUTUSAN: OBSERVATION**
- Boleh standby di ruang tunggu, pemeriksaan ulang 30 menit

#### MERAH (Tidak Boleh Masuk)
- Demam tinggi (>38°C)
- Gejala respirasi berat
- Kondisi medis akut
- Clearance tidak valid/expired
- **KEPUTUSAN: REJECTED**
- Langsung arahkan ke ruang observasi/rujukan

### 4.5. Ruang Observasi Cepat
Untuk kategori KUNING dan MERAH:

1. Pisahkan dari area utama
2. Pemeriksaan lebih detail:
   - Tanda vital lengkap
   - Anamnesa singkat
   - Cek riwayat kesehatan di sistem

3. Keputusan:
   - Jika membaik: boleh masuk dengan monitoring
   - Jika tidak membaik/memburuk: rujuk ke faskes

### 4.6. Dokumentasi
1. Setiap entry check dicatat di sistem:
   - Waktu check
   - Lokasi checkpoint
   - Suhu dan keluhan
   - Kategori triage
   - Keputusan (Approved/Observation/Rejected)
   - Nama petugas

2. Laporan real-time ke command center

### 4.7. Komunikasi
1. Personel yang ditolak:
   - Jelaskan alasan dengan sopan
   - Berikan arahan alternatif
   - Catat untuk follow-up

2. Koordinasi dengan:
   - Pos kesehatan utama
   - Komando/pimpinan
   - Tim keamanan

## 5. Penanggung Jawab
- **Supervisor Pos**: Koordinasi dan pengambilan keputusan
- **Petugas Medis**: Pemeriksaan dan assessment
- **Admin**: Input data dan dokumentasi

## 6. Catatan Khusus
- Untuk VIP: koordinasi dengan liaison officer
- Untuk kondisi darurat: hubungi dokter on-call
- Jaga kerahasiaan data medis personel

## 7. Peralatan Backup
- Termometer manual
- Formulir kertas
- Jalur komunikasi alternatif (HT/radio)

## 8. Indikator Keberhasilan
- 100% personel melewati entry check
- Waktu tunggu <5 menit per personel
- Tidak ada kasus penyakit menular masuk area
- Dokumentasi lengkap dan akurat
